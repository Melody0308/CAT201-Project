# CAT201 Java E-Commerce Project

## Project Description
This project is a simple Java-based e-commerce web application developed for the
CAT201 Integrated Software Development Workshop.

The project focuses on demonstrating basic Java web application development using
Servlet and JSP, as well as teamwork and version control with GitHub.

---

## Technologies Used
- Java Servlet
- JSP
- HTML / CSS
- GitHub (Version Control)


<h2>Project Structure</h2>
<p>The project follows a standard Java Web application structure:</p>

<pre>
src/
└── main/
    ├── java/
    │   └── com/
    │       └── cat201/
    │           ├── servlet/
    │           ├── model/
    │           └── dao/
    └── webapp/
        └── WEB-INF/
</pre>
